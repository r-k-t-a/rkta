## Validator

Validate form inputs

## Arguments

- schema - ajv validation schema, js object
- options - ajv option

[AJV](https://www.npmjs.com/package/ajv)

## Usage
```
import { Form, makeValidator } from '@rkta/form';

const schema = {
  type: 'object',
  properties: {
    login: { type: 'string' }
  },
  required: ['login']
}

const validate = makeValidator(schema, { useDefaults: false });

<Form validate={validate} onFormSubmit={(form) => { console.log(form) }}>
  <Input name="login" />
</Form>

```
