Provider

Use Provider to wrap your application and create @rkta/ui context.

Props

theme
Rktatheme | undefined
getElement
Resolver
required

Example

import Provider as UiProvider from '@rkta/ui/Provider';
<UiProvider theme={{ color: { customColor: '#fff' } }}>
...
</UiProvider>

Context Hook

Grab provider context inside your hooks.

import { useProviderContext } from '@rkta/ui/Provider';
function MyComponent() {
  const { theme, replaceTheme } = useProviderContext();
  ...
}
