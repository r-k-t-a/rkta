# Divider

Divider is a horizontal rule (line) which usually separates content blocks

## Examples

<Playground>
  <ScrollBox x css={{ display: 'flex' }}>
    {Range(0, 50).map(item => (
      <Paper bgColor="primary" center size={50} css={{ margin: 4, flexShrink: 0 }}>
        {item}
      </Paper>
    ))}
  </ScrollBox>
</Playground>
