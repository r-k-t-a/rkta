# Placeholder

Wrap texts to create placeholders

## Default usage

<Playground>
  <Text caption element="div">
    Before load:
  </Text>
  <Text>
    <Placeholder>{undefined}</Placeholder>
  </Text>
  <Divider />
  <Text caption element="div">
    After load:
  </Text>
  <Text>
    <Placeholder>Some text</Placeholder>
  </Text>
</Playground>

## Set width and color

<Playground>
  <Text caption element="div">
    Before load:
  </Text>
  <Text color="primary">
    <Placeholder width="7em">{undefined}</Placeholder>
  </Text>
  <Divider />
  <Text caption element="div">
    After load:
  </Text>
  <Text color="primary">
    <Placeholder width="7em">Some text longer</Placeholder>
  </Text>
</Playground>
