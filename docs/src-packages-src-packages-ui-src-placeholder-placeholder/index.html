Placeholder

Wrap texts to create placeholders

Default usage

Before load:
After load:
Some text
<Text caption element="div">
  Before load:
</Text>
<Text>
  <Placeholder>{undefined}</Placeholder>
</Text>
<Divider />
<Text caption element="div">
  After load:
</Text>
<Text>
  <Placeholder>Some text</Placeholder>
</Text>

Some text longer

  Before load:
</Text>
<Text color="primary">

</Text>

</Text>
