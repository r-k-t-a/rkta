# ArrowKeys

ArrowKeys enable keyboard navigation for list and nav components.

## Note

ArrowKeys has a few side-effects:

- `aria-selected` is injected to all `validType` elements
- `ref` is injected to selected element

## Import

```javascript
import { ArrowKeys } from '@rkta/ui';
```

## Examples

<Playground>
  {() => {
    const list1 = [1, 2, 3];
    const list2 = [4, 5];
    return (
      <List rize={1} sans>
        <ArrowKeys vertical>
          <ListTitle>Menu List</ListTitle>
          <Divider horizontalSpace={16} />
          {list1.map(item => (
            <ListButton key={item}>Option {item}</ListButton>
          ))}
          <Divider horizontalSpace={16} />
          {list2.map(item => (
            <ListButton key={item}>Option {item}</ListButton>
          ))}
        </ArrowKeys>
      </List>
    );
  }}
</Playground>
