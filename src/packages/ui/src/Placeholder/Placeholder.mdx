# Placeholder

Wrap texts to create placeholders

## Default

<Playground>
  <Text caption element="div">
    Before load:
  </Text>
  <Text>
    <Placeholder>{undefined}</Placeholder>
  </Text>
  <Divider />
  <Text caption element="div">
    After load:
  </Text>
  <Text>
    <Placeholder>Some text</Placeholder>
  </Text>
</Playground>

## Multiline

<Playground>
  <Text>
    <Placeholder />
    <Placeholder />
    <Placeholder width="56%" />
  </Text>
</Playground>

## Set width and color

<Playground>
  <Text caption element="div">
    Before load:
  </Text>
  <Text color="primary">
    <Placeholder inline width="7em">
      {undefined}
    </Placeholder>
  </Text>
  <Divider />
  <Text caption element="div">
    After load:
  </Text>
  <Text color="primary">
    <Placeholder width="7em">Some longer text</Placeholder>
  </Text>
</Playground>
